# OsuBestplayDownloader

<h2>如何使用</h2>
一. 从Release界面下载osu-bp.zip, 解压 (注意连同oauth文件一并解压)

</br>

二. 配置OAuth

1.打开程序后，点击"配置OAuth"
</br>
2.进入osu!官网 (https://osu.ppy.sh)
</br>
3.登录，随后点击头像，点击设置，打开设置界面
</br>
4.拉到最底下，找到"开放授权"
</br>
5.点击"新的 OAuth 应用"
</br>
6.弹出的窗口中"应用名称"随便填，"应用回调链接"不填
</br>
7.点击"注册应用程序"
</br>
8.点击"显示客户端密钥"
</br>
9.程序内填入ID及密钥，点击"应用设置"，关闭窗口
</br>
